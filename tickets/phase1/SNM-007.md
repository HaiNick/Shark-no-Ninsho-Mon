# SNM-007 · HIGH

**Expand SSRF blocklist for cloud metadata IPs**

|-------|-------|
| **Priority** | HIGH |
| **Phase** | 1 — Immediate (Before Production) |
| **File** | `app/routes_db.py:202-219` |

## Problem

Only `169.254.169.254` is explicitly blocked. IPv6 link-local, Alibaba Cloud, and other metadata IPs are open.

## Fix

```python
BLOCKED_IPS = {
    '169.254.169.254',  # AWS/GCP/Azure (IPv4)
    'fd00:ec2::254',    # AWS (IPv6)
    '100.100.100.200',  # Alibaba Cloud
    '169.254.0.23',     # OpenStack
}

if ip_obj.is_link_local:
    raise ValueError("Link-local IPs are not allowed")
if str(ip_obj) in BLOCKED_IPS:
    raise ValueError("Cloud metadata IPs are not allowed")
if ip_obj.is_reserved:
    raise ValueError("Reserved IPs are not allowed")
```
