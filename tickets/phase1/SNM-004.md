## Problem

`DEV_MODE=true` silently disables all auth. No warning if accidentally shipped in production.

## Fix

```python
if os.environ.get('DEV_MODE', '').lower() == 'true':
    logger.critical("=" * 60)
    logger.critical("DEV_MODE ENABLED — ALL AUTHENTICATION BYPASSED")
    logger.critical("NEVER use in production!")
    logger.critical("=" * 60)
    if os.path.exists('/.dockerenv'):
        logger.error("DEV_MODE is active inside Docker — this is likely a mistake!")
    if os.environ.get('OAUTH2_PROXY_CLIENT_ID'):
        logger.error("DEV_MODE active with OAuth configured — unsafe!")
```

- [x] CRITICAL log banner appears at container startup when `DEV_MODE=true`
- [x] Warning logged when DEV_MODE is active inside Docker
- [x] Warning logged when DEV_MODE is active alongside OAuth credentials
- [x] No log noise when `DEV_MODE` is unset or `false`
