# SNM-002 · HIGH

**Secure / disable setup wizard after initial setup**

| Field | Value |
|-------|-------|
| **Priority** | HIGH |
| **Phase** | 1 — Immediate (Before Production) |
| **File** | `setup-wizard.py:833` |
| **Status** | ⬜ Open |

---

## Problem

Wizard binds to `0.0.0.0:8080` unauthenticated — exposes secrets, email lists, and Docker control to anyone on the LAN.

## Fix (Option A — localhost only)

```python
app.run(host='127.0.0.1', port=8080, debug=False)
```

## Fix (Option B — single-use token)

```python
import secrets
SETUP_TOKEN = secrets.token_urlsafe(32)
print(f"[SETUP] Open: http://localhost:8080?token={SETUP_TOKEN}")

@app.before_request
def check_token():
    if request.endpoint != 'static':
        if request.args.get('token') != SETUP_TOKEN:
            return jsonify({'error': 'Invalid token'}), 403
```

Also add: print a warning to run `docker compose down` after setup completes.
