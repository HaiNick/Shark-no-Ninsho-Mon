**Validate OAuth2 header source in Flask**

## Problem

Flask trusts `X-Forwarded-Email` blindly. A direct `curl -H "X-Forwarded-Email: admin@x.com" http://flask:8000` bypasses all auth.

## Fix (Recommended — network isolation)

```yaml
# docker-compose.yml: remove public port for Flask
# ports:
#   - "8000:8000"   ← REMOVE
```

## Fix (Defence-in-depth — shared secret)

```python
PROXY_SECRET = os.environ.get('OAUTH_PROXY_SHARED_SECRET')

def validate_proxy_request():
    if is_dev_mode():
        return True
    sig = request.headers.get('X-Auth-Request-Signature')
    if not sig or not PROXY_SECRET:
        return False
    expected = hmac.new(PROXY_SECRET.encode(), request.path.encode(), hashlib.sha256).hexdigest()
    return hmac.compare_digest(sig, expected)
```
