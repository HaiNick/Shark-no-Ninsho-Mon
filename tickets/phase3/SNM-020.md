## Problem

Raw `localStorage` usage without expiration — stale data persists indefinitely.

## Fix

Replace raw `localStorage` with a `CacheManager` wrapper that stores `{ data, timestamp }` and respects a configurable `maxAge` (default 300s). Add `invalidate(pattern)` for cache busting after mutations.

## PR Checkpoints

- [x] `CacheManager` wrapper implemented in `utils.js`
- [x] Stale entries (older than `maxAge`) automatically evicted on read
- [x] `invalidate(pattern)` called after all route create/update/delete operations
- [x] No regressions in dashboard data loading behaviour
