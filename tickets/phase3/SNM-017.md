# SNM-017 · LOW

**Extract magic numbers/strings to a constants file**

| Field | Value |
|-------|-------|
| **Priority** | LOW |
| **Phase** | 3 — Long Term (Next Quarter) |
| **File(s)** | Multiple |
| **Status** | ⬜ Open |

---

## Problem

Hardcoded magic numbers and strings are scattered throughout the codebase, reducing maintainability.

## Fix

Create `app/constants.py` with classes: `Limits`, `RateLimits`, `Defaults`. Replace all hardcoded values throughout the codebase.

## PR Checkpoints

- [ ] `app/constants.py` created with `Limits`, `RateLimits`, `Defaults` classes
- [ ] All hardcoded rate limit strings replaced with constants
- [ ] All hardcoded timeout/interval values replaced with constants
- [ ] No functional behaviour changed
