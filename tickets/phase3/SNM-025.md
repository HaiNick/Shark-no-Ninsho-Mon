# SNM-025 · LOW

**Set up automated security scanning in CI**

| Field | Value |
|-------|-------|
| **Priority** | LOW |
| **Phase** | 3 — Long Term (Next Quarter) |
| **File** | `.github/workflows/` |
| **Status** | ⬜ Open |

---

## Problem

No automated security scanning in CI pipeline. Vulnerabilities may go undetected.

## Fix

```yaml
- name: Security scan
  run: |
    pip install safety bandit
    safety check
    bandit -r app/ -ll
```

## PR Checkpoints

- [ ] GitHub Actions workflow file created or updated
- [ ] `safety check` runs and passes on current dependencies
- [ ] `bandit -r app/ -ll` runs and reports no HIGH findings
- [ ] Workflow triggers on push and pull_request to `main`
