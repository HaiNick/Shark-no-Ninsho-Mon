**Set up automated security scanning in CI**

## Problem

No automated security scanning in CI pipeline. Vulnerabilities may go undetected.

## Fix

```yaml
- name: Security scan
  run: |
    pip install safety bandit
    safety check
    bandit -r app/ -ll
```

## PR Checkpoints

- [x] GitHub Actions workflow file created or updated
- [x] `safety check` runs and passes on current dependencies
- [x] `bandit -r app/ -ll` runs and reports no HIGH findings
- [x] Workflow triggers on push and pull_request to `main`
