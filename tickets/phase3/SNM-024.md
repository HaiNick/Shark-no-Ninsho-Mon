| Field | Value |
|-------|-------|
| **Priority** | LOW |
| **Phase** | 3 — Long Term (Next Quarter) |

## Problem

No security headers (HSTS, X-Content-Type-Options, X-Frame-Options) set by the Flask app.

## Fix

```python
from flask_talisman import Talisman
Talisman(app, content_security_policy=False)  # Enable CSP incrementally
```

## PR Checkpoints

- [x] `Flask-Talisman` initialised in `app.py`
- [x] Response headers include `Strict-Transport-Security`
- [x] Response headers include `X-Content-Type-Options: nosniff`
- [x] Response headers include `X-Frame-Options: SAMEORIGIN`
- [x] Dashboard still loads correctly after Talisman enabled
