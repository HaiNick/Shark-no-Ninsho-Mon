**Add comprehensive type hints throughout Python codebase**

| Field | Value |
|-------|-------|
| **Priority** | LOW |
| **Phase** | 3 — Long Term (Next Quarter) |

## Problem

Functions lack type annotations, making the codebase harder to maintain and reason about.

## Fix

Add `from typing import Dict, List, Optional, Tuple, Any` imports and annotate all function signatures. Run `mypy app/` to verify.

## PR Checkpoints

- [ ] All public functions in `app.py`, `routes_db.py`, `caddy_manager.py`, `config.py` have type hints
- [ ] `mypy app/` runs without errors
- [ ] No runtime behaviour changed
