## Problem

Multiple Flask workers can call `sync()` simultaneously — race condition in Caddy config updates.

## Fix

Add `threading.Lock()` around sync. Enforce minimum 1s between syncs via debounce.

## PR Checkpoints

- [x] `threading.Lock()` guards all `sync()` calls
- [x] Minimum 1s debounce between syncs enforced
- [x] Lock acquisition timeout of 30s prevents deadlock
- [x] Sync success and failure both logged with route count
