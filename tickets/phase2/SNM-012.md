## Problem

Multiple Flask workers can call `sync()` simultaneously — race condition in Caddy config updates.

## Fix

Add `threading.Lock()` around sync. Enforce minimum 1s between syncs via debounce.

## PR Checkpoints

- [ ] `threading.Lock()` guards all `sync()` calls
- [ ] Minimum 1s debounce between syncs enforced
- [ ] Lock acquisition timeout of 30s prevents deadlock
- [ ] Sync success and failure both logged with route count
