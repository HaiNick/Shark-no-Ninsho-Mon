## Problem

TinyDB is file-based JSON with in-process locking only. Not suitable for production write concurrency.

## Fix

```python
conn = sqlite3.connect('routes.db', isolation_level='IMMEDIATE')
conn.execute('PRAGMA journal_mode=WAL')
```

Wrap all ops in context-managed connections with commit/rollback. Add indexes on `path` and `enabled`.
